# Участники проекта (Contributors)

Проект является развитием исследовательской работы по адаптации архитектуры Video-RAG для русскоязычного видеоконтента.

### Основной разработчик
* **Ляра Максим Витальевич** (maxlyara1)
  - Проектирование и реализация универсального модульного конвейера поиска (`VideoRAGPipeline`).
  - Интеграция локальных моделей ASR (Whisper), OCR (EasyOCR) и визуальных описаний (BLIP-base + spaCy).
  - Настройка схемы многоканального индексирования и векторизации с локальной моделью `Qwen3-Embedding-0.6B`.
  - Интеграция векторной базы данных Qdrant (создание и фильтрация 3 независимых векторных коллекций).
  - Реализация алгоритмов временной группировки интервалов и слияния результатов (RRF, Max-per-Modality, Sum).
  - Разработка асинхронного серверного интерфейса (FastAPI) с трансляцией состояния выполнения по SSE и адаптивного веб-интерфейса.
  - Настройка окружения Docker и разработка автоматизированной оценки качества поиска (`scripts/run_benchmark.py`).
