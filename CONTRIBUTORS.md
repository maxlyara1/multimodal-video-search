# Участники проекта (Contributors)

Проект является развитием исследовательской работы по адаптации архитектуры Video-RAG для русскоязычного видеоконтента.

### Основной разработчик
* **Ляра Максим Витальевич** (maxlyara1)
  - Проектирование и реализация универсального модульного пайплайна поиска (`VideoRAGPipeline`).
  - Интеграция локальных моделей ASR (Whisper), OCR (EasyOCR) и Visual Captions (BLIP-base + spaCy).
  - Настройка схемы многоканального индексирования и векторизации с локальной моделью `Qwen3-Embedding-0.6B`.
  - Интеграция векторной СУБД Qdrant (создание и фильтрация 3 независимых векторных коллекций).
  - Реализация алгоритмов временной группировки интервалов и взвешенного слияния (weighted score fusion, RRF, Max-Norm).
  - Разработка асинхронного REST API (FastAPI) с SSE-стримингом логов и адаптивного веб-интерфейса MVP.
  - Настройка окружения Docker и разработка автоматизированного бенчмарка (`scripts/run_benchmark.py`).
