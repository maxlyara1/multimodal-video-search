# Уведомление о происхождении и лицензиях (Notice)

Система мультимодального поиска по видео (`multimodal-video-search`) является самостоятельной реализацией, вдохновленной концепциями и архитектурными принципами научной статьи:

> **Video-RAG: Visually-aligned Retrieval-Augmented Long Video Comprehension**  
> *Ссылка на публикацию: arXiv:2411.13093*  
> *Исходный репозиторий: https://github.com/Leon1207/Video-RAG-master*

### Отличие реализации:
1. Данный репозиторий представляет собой независимую прикладную систему, ориентированную на универсальный конвейер обработки и поддержку русскоязычных видеоматериалов.
2. В отличие от оригинальной работы, которая передает извлеченные тексты (речь, распознанный текст на экране, обнаруженные объекты) вместе с видеокадрами в зрительно-языковую модель, данная реализация заменяет распознавание объектов на описания кадров моделью BLIP-base и использует текстовую генерацию ответов через Gemini API по собранному текстовому контексту (без непосредственной передачи видео или кадров на генеративном этапе) с локальным извлечением признаков:
   - **ASR**: Локальный Whisper (конфигурация turbo) с параллельной нарезкой длинных аудиозаписей.
   - **OCR**: Локальный EasyOCR с поддержкой кириллицы.
   - **Описания кадров**: BLIP-base, после чего библиотека spaCy выделяет упомянутые объекты и простые синтаксические связи между ними.
3. Интегрирована легковесная модель `Qwen3-Embedding-0.6B` для получения векторных представлений текстовых данных (расшифровки речи, текста с экрана и текстовых описаний кадров).
4. Добавлен локальный веб-интерфейс плеера (серверная часть на FastAPI и интерфейс на HTML, CSS и JavaScript) с передачей состояния выполнения через SSE и поддержкой перетаскивания файлов.
5. Реализованы способы объединения результатов поиска из разных источников:
   - **RRF (Reciprocal Rank Fusion)**: ранговое слияние результатов.
   - **Max-per-Modality**: объединение оценок косинусного сходства по максимальному значению модальности.
