# История проекта (Project History)

Проект прошел путь эволюции от первичной теоретической репликации научной статьи до универсальной доменно-независимой системы поиска по видеоконтенту.

### Этап 1: Теоретическое исследование (Апрель 2026)
* **Цель**: Изучение и проверка применимости подходов, описанных в научной статье *«Video-RAG: Visually-aligned Retrieval-Augmented Long Video Comprehension»*.
* **Результаты**:
  - Были подготовлены тестовые русскоязычные видеоматериалы.
  - Осуществлен первый запуск и оценка базового распознавания речи (ASR) на тестовом наборе.

### Этап 2: Хакатонный прототип (Май 2026)
* **Цель**: Реализация быстрого прототипа для поиска по видео в рамках хакатона.
* **Результаты**:
  - Была создана первая ASR-only система семантического поиска.
  - Пайплайн был глубоко интегрирован со специфическими закрытыми метаданными и API видеосервиса Okko.
  - Решение не имело самостоятельного пользовательского интерфейса и функционировало в виде консольных скриптов.

### Этап 3: Универсальный Video-RAG и MVP (Июнь - Июль 2026)
* **Цель**: Полное очищение кодовой базы от хакатонного доменного кода, переход к универсальной архитектуре и построение веб-ориентированного MVP.
* **Результаты**:
  - Пайплайн переписан в виде независимого оркестратора `VideoRAGPipeline`.
  - Добавлены две новые модальности: OCR (EasyOCR с поддержкой русского языка) и Visual Captions (генерация описаний сцен с BLIP-base и извлечение сущностей через spaCy).
  - В качестве векторного хранилища интегрирована СУБД Qdrant (3 коллекции для ASR, OCR и визуальных признаков) с локальным индексированием на Apple Silicon GPU.
  - Построен асинхронный бэкенд на FastAPI с SSE-стримингом логов и разработан легкий адаптивный веб-интерфейс плеера на чистом CSS/JS.
  - Подготовлен автоматизированный бенчмарк для сравнения стратегий поиска и проведено детальное тестирование.
